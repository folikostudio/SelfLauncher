What is SelfLauncher

SelfLauncher is not a ready-made launcher with predefined behavior. It is a universal platform for building custom launchers for any purpose.

Core Principles

• Minimal, neutral core
• Command-based action model
• Configuration as the primary source of logic
• Full modularity and extensibility
• UI as an external, optional layer

Philosophy

SelfLauncher rejects the idea of a single “correct” way to use a launcher. The same executable can play completely different roles, depending on configuration and connected components.

What It Is Not

❌ a universal ready-to-use launcher
❌ a rigid, opinionated tool
❌ a system with fixed usage scenarios
